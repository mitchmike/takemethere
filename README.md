# TakeMeThere

A smarter way to navigate Melbourne's metro train network.

## What it does

TakeMeThere gives you a live, map-based view of Melbourne Metro trains and uses real-time GTFS data to help you plan journeys that are actually informed by what's happening on the network right now — not just what the timetable says.

### Key features (planned)

- **Live train map** — see trains moving on your chosen lines in real time
- **Smart journey planning** — accounts for late-running services, surfaces express trains, and times changeovers so you don't miss a connection
- **Delay-aware notifications** — better than Metro Notify: alerts tuned to your actual journey, not just a line status
- **Better UX than the alternatives** — a cleaner, faster experience compared to the PTV app, Metro's timetable site, and Google Maps

## Data sources

- [GTFS static feed](https://www.ptv.vic.gov.au/footer/data-and-reporting/datasets/gtfs/) — timetables, stops, routes
- [GTFS-RT feed](https://www.ptv.vic.gov.au/footer/data-and-reporting/datasets/gtfs/) — real-time vehicle positions and trip updates
- Additional sources TBD as the project matures
